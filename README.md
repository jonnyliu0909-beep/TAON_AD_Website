# TAON Advertising Studio Website

TAON Advertising Studio 官方网站 - 多语言支持的企业网站

## 🌐 语言支持

- English (英语)
- Español (西班牙语)
- Français (法语)
- 中文 (简体中文)
- 繁體中文 (繁体中文)
- 日本語 (日语)
- 한국어 (韩语)

## 📁 项目结构

```
TAON_AD_Website/
├── index.html          # 首页
├── contact.html        # 联系页面
├── services.html       # 服务页面
├── archive.html        # 案例页面
├── single.html         # 关于页面
├── leave-message.html  # 留言板页面
├── css/               # 样式文件
├── js/                # JavaScript 文件
├── images/            # 图片资源
├── i18n/              # 国际化翻译文件
├── font-awesome/      # Font Awesome 图标库
└── owl-carousel/      # 轮播组件
```

## 🚀 部署

本网站可以部署到 GitHub Pages 或其他静态网站托管服务。

详细部署指南请参考：[GITHUB_PAGES_DEPLOYMENT.md](GITHUB_PAGES_DEPLOYMENT.md)

## ✨ 功能特性

- ✅ 响应式设计，支持移动端和桌面端
- ✅ 多语言支持（7种语言）
- ✅ 留言板功能（支持本地存储和服务器存储）
- ✅ SEO 优化
- ✅ 社交媒体集成
- ✅ 联系表单

## 📝 注意事项

- 留言板功能在 GitHub Pages 上使用 localStorage（本地存储）
- 如需服务器端存储，请使用支持 PHP 的托管服务或第三方 API
